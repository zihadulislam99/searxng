Redis DB¶

Implementation of the redis client (redis-py).

This implementation uses the redis: setup from settings.yml. A redis DB connect can be tested by:

>>> from searx.shared import redisdb
>>> redisdb.init()
True
>>> db = redisdb.client()
>>> db.set("foo", "bar")
True
>>> db.get("foo")
b'bar'
>>>
